## Description
`Milky2018/wgsl/common` currently owns compose options, preprocess outputs, import descriptors, source registries, diagnostics for composition, symbol redirects, and Bevy-oriented default shader definitions. This reverses conceptual dependency: compatibility policy changes force a WGSL Core release.

## Design
Delete the mixed-purpose `wgsl/common` interface. Keep official directive and language diagnostic contracts in an official WGSL package. Move preprocess, import, compose, export, source catalog, registry, and compatibility diagnostics into public packages owned by `Milky2018/moon_wgsl_naga_oil`. Move project-profile defaults into an explicit naga-oil profile module; default constructors should use language-neutral empty values unless a profile is selected.

Update the facade to use and, where appropriate, re-export the new owning types without keeping aliases in WGSL Core.

## Acceptance Criteria
- [x] `Milky2018/wgsl/common` is removed or contains only official WGSL language contracts with a precise package name.
- [x] Compose, preprocess, import, export, source registry, source-map, and symbol redirect types are owned by `moon_wgsl_naga_oil`.
- [x] Bevy or project-profile shader defaults are not part of WGSL Core or generic compose defaults.
- [x] No compatibility alias remains in WGSL Core.
- [x] Public examples and tests import each contract from its owning module or the facade.
- [x] Module checks, workspace tests, architecture guardrails, and strict external compose parity pass.
