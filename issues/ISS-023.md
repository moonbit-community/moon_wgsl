## Description
Create `Milky2018/moon_wgsl_naga_oil` as the owner of naga-oil-compatible preprocessing and composition.

## Design
The module owns naga-oil directives, shader defs, import substitution, metadata, resolver behavior, compose graph construction, symbol graph binding, final name allocation, virtual overrides, and compose emission. It may depend on `Milky2018/moon_wgsl_naga` for explicitly Naga-shaped writer behavior, but it owns naga-oil import and composition semantics.

## Acceptance Criteria
- [x] Preprocess and compose implementation moves to `modules/moon_wgsl_naga_oil`.
- [x] Composer semantics are driven by `ComposeGraph`, `SymbolGraph`, `FinalNameTable`, and `EmissionPlan`.
- [x] Source edit backends consume already-bound plans and do not decide reachability, aliases, or final names.
- [x] naga-oil preprocess and compose parity tests run against this module.
- [x] The module is publishable with a clear API boundary.
