## Description
Decompose the old `transform` package into generic WGSL analysis and naga-oil compatibility rewrite responsibilities.

## Design
Generic declaration cataloging, AST dependency analysis, and language-level origin extraction belong in `Milky2018/wgsl`. Import/alias/final-name rewrite plans, virtual override handling, and source edit backends belong in `Milky2018/moon_wgsl_naga_oil` unless a smaller source edit utility is required internally by WGSL diagnostics.

## Acceptance Criteria
- [x] Generic WGSL analysis lives under `modules/wgsl`.
- [x] Binding rewrite plans and virtual override behavior live under `modules/moon_wgsl_naga_oil`.
- [x] `Source Edit Backend` is not exposed as a WGSL semantic API.
- [x] No package combines WGSL analysis with naga-oil rewrite policy.
- [x] Existing transform tests are split by ownership and pass.
