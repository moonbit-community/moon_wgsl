## Description
Replace the single-module CI shape with module-local fast gates and workspace integration gates.

## Design
Each module gets local `fmt`, `info`, `check`, and test gates for the behavior it owns. Workspace gates run heavyweight external corpora, CTS/Naga differential validation, naga-oil byte parity, wgpu validation tooling, and publish packaging checks.

## Acceptance Criteria
- [ ] CI has separate fast gates for `wgsl`, `moon_wgsl_naga`, `moon_wgsl_naga_oil`, and `moon_wgsl`.
- [ ] WGSL Core gates include CTS and Naga parser/validator oracle checks.
- [ ] Naga compatibility gates include writer/order/name parity checks.
- [ ] naga-oil compatibility gates include preprocess/compose parity checks.
- [ ] Workspace integration gates own large corpora, oracle tools, and wgpu validation.
- [ ] CI failure ownership is clear from the failing job name.
