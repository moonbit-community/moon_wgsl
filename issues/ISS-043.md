## Description
Move the maintained `Milky2018/moon_wesl` implementation into this workspace,
then replace its duplicate official WGSL frontend semantics with the
`Milky2018/wgsl` boundary before publishing from this repository and
deprecating the former repository.

## Design
Import only the source repository's Git-tracked tree into
`modules/moon_wesl`. Preserve behavior in the first stage and keep WESL outside
the existing synchronized compatibility release line. In the second stage,
make WESL depend on WGSL Core for official WGSL parsing and validation while it
continues to own WESL syntax, module resolution, conditional compilation, and
assembly. Review and shrink the public API before the first release from this
workspace.

The imported `README.md` symlink is a temporary snapshot-preservation exception.
It must be replaced or deliberately supported before this issue closes.
