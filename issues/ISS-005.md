## Description
After recent arena fixes, the external naga-oil compose gate passes with known drift allowed: `cases=150 comparable=149 oracle-blocked=1 writer-exact=149 writer-drift=0 byte-exact=105 byte-drift=44`. These byte drift cases remain known incompatibilities. They should be reduced by solving shared root causes discovered through module, body, and byte-fragment traces.

## Design
Do not select files only because they look small. Group drift cases by trace divergence class, fix one class at a time, and update `external_naga_oil_compose_byte_drift.tsv` and `external_naga_oil_compose_drift_taxonomy.tsv` only from observed gate output.

- [ ] `bash tools/check_external_naga_oil_drift_taxonomy.sh` and `moon test` pass after each class fix.
