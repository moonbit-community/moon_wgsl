# ISS-005: Reduce remaining byte drift by trace-derived classes

## Metadata

## Description
After recent arena fixes, the external naga-oil compose gate passes with known drift allowed and the observed byte drift set is down to 15 cases. These byte drift cases remain known incompatibilities. They should be reduced by solving shared root causes discovered through module, body, and byte-fragment traces.

## Design
Do not select files only because they look small. Group drift cases by trace divergence class, fix one class at a time, and update `external_naga_oil_compose_byte_drift.tsv` and `external_naga_oil_compose_drift_taxonomy.tsv` only from observed gate output.

## Acceptance Criteria
- [ ] Each fixed drift class references the trace issue that identified its root cause.
- [ ] Every manifest update comes from `_build/parity/external_naga_oil_compose/byte_failures.tsv`.
- [ ] `MOON_WGSL_ALLOW_KNOWN_DRIFT=1 bash tools/check_external_naga_oil_compose_parity.sh` passes after each class fix.
- [ ] `bash tools/check_external_naga_oil_drift_taxonomy.sh` and `moon test` pass after each class fix.
