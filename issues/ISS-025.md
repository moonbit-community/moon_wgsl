## Description
Create `Milky2018/moon_wgsl` as the thin user-facing facade over the workspace modules.

## Design
The facade exposes user-facing preprocess and compose convenience APIs only. It must not re-export WGSL parser/AST/IR internals, Naga compatibility internals, source edit backends, symbol graphs, or writer plans. Legacy subpackage compatibility is intentionally dropped.

## Acceptance Criteria
- [x] `modules/moon_wgsl` exposes only the intended public facade API.
- [x] README examples use the facade for normal preprocess/compose workflows.
- [x] WGSL parser/AST/IR users are directed to `Milky2018/wgsl`.
- [x] Naga compatibility users are directed to `Milky2018/moon_wgsl_naga`.
- [x] naga-oil compatibility users are directed to `Milky2018/moon_wgsl_naga_oil`.
- [x] No legacy internal subpackage forwarding exists.
