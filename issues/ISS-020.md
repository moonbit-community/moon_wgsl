## Description
Move the language-level WGSL packages into `modules/wgsl` as the complete official WGSL frontend.

## Design
`Milky2018/wgsl` owns WGSL language facts: lexer, parser, AST, semantic validation, semantic IR, generic writer, and generic WGSL analysis. It must not depend on `moon_wgsl`, `moon_wgsl_naga`, `moon_wgsl_naga_oil`, or future WESL modules.

## Acceptance Criteria
- [x] `modules/wgsl` contains the WGSL lexer, parser, AST, semantic validation, and semantic IR packages.
- [x] `modules/wgsl` exposes parser/AST/IR APIs under `Milky2018/wgsl/*`, not old `Milky2018/moon_wgsl/*` paths.
- [x] `modules/wgsl` does not import any compatibility module.
- [x] Official WGSL parser/validator tests run from `modules/wgsl`.
- [x] Existing WGSL core package tests are migrated and passing.
