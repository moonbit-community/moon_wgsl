## Description
Create `Milky2018/moon_wgsl_naga` as a first-class publishable module for Naga-compatible behavior.

## Design
This module owns Naga-compatible module views, declaration ordering, expression arenas, temporary naming, writer policy, validation adapters, and trace tooling. It depends on `Milky2018/wgsl` but not on the `moon_wgsl` facade.

## Acceptance Criteria
- [ ] Naga writer/order/name allocation code lives in `modules/moon_wgsl_naga`.
- [ ] Naga validation adapters and trace tooling live in `modules/moon_wgsl_naga`.
- [ ] `modules/moon_wgsl_naga` is publishable with its own README/API boundary.
- [ ] Naga writer parity tests run against this module.
- [ ] No Naga compatibility implementation remains in `Milky2018/wgsl`.
